Statistics

This section makes reference to the files in the /stats folder.

R Code

The dataAnalysis.r file contains the R code used to load and analyze the information obtained from the project code.

CSV Files

These are the files obtained from the project code with the features of the files listed.

EntrData.csv

Contains the basic statistics about the files and if it is a virus or not (Type). The basic statistics are:

  • File - Full file entropy
  • Min - The entropy of the block with the lowest entropy.
  • 1Q - The first quartile of file blocks.
  • Median - Guess
  • Mean - Guess
  • 3Q - Guess
  • Max - Guess
  • Type - 0: if not a virus. 1: if the file is a virus.

struct.csv

Contains the features obtained from the mean change point analysis.
The fetures in this file are:

  • maxJumpSize
  • minJumpSize
  • maxLongestRunPct - The percent of the longest segment vs the sum of all segments.
  • meanLongestRun - The mean entropy of the longest segment.
  • Segment_Count - How many segments.

alphaFT.csv

Has the features extracted from the Detrended fluctuation and Fourier transform Analysis.
The fetures in this file are:

  • Alpha - The DFA statistic.
  • MagN - The magnitude of Nth "freqency"
  • RatioN - The ratio MagN/Mag0
  • Pitch - The Harmonic with the highest magnitude
